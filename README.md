handy
=====
a web framework for golang

### 介绍
用 Go 实现的一个简单的 MVC 模式框架，目前支持：

* 路由/Restful(Route)
* 控制器(handler)
* 视图(templates)
* 表单(form)
* 静态文件(static)

### 安装
请确保Go环境已经安装，如未安装请参考 [Go 环境安装](http://golang.org/doc/install.html)，请安装最新版。

安装web框架
> go get github.com/web-framework/handy

### 使用
> import "github.com/web-framework/handy"
